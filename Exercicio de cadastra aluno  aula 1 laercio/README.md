# 📚 Exercício: Cadastro de Alunos (Aula 1)

Este exercício foi desenvolvido em **C#** como prática de lógica de programação e manipulação de dados no console.

O programa permite **cadastrar alunos**, registrar **idade**, **curso**, **notas** e calcular automaticamente a **média final**, informando se o aluno foi **aprovado ou não**.

---

## 🚀 Funcionalidades

O programa permite:

- 📌 Cadastrar alunos
- 📌 Informar idade e curso
- 📌 Registrar 3 notas
- 📌 Calcular a média automaticamente
- 📌 Verificar se o aluno foi aprovado
- 📌 Armazenar alunos em listas
- 📌 Mostrar todos os alunos cadastrados

---

## 🧠 Conceitos praticados

Neste exercício foram utilizados conceitos importantes de C#:

- `List<T>`
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if`)
- Validação de entrada do usuário
- Funções
- Manipulação de dados no console

---

## ▶️ Como executar

1. Abra o projeto no **Visual Studio** ou **VS Code**
2. Execute o programa
3. Insira os dados solicitados no terminal
